Chatwoot WooCommerce Orders Panel

A lightweight integration that displays a customer's WooCommerce orders directly inside the Chatwoot conversation sidebar.

Instead of leaving Chatwoot, opening WordPress admin, navigating to WooCommerce and searching for the customer, agents can instantly view:

Customer order history
Order status and totals
Purchased products
Product metadata and custom item attributes
Direct links to edit orders in WooCommerce
Expandable order details inside Chatwoot
Features

✅ Embedded order history inside Chatwoot

✅ Uses Chatwoot Contact Custom Attributes (wp_user_id)

✅ WooCommerce REST endpoint for retrieving customer orders

✅ Supports product metadata and variation attributes

✅ Expandable order cards with itemized details

✅ One-click access to WooCommerce order editing

✅ No external services required

✅ Self-hosted and privacy-friendly

Requirements
Chatwoot (self-hosted)
WordPress
WooCommerce
A custom contact attribute in Chatwoot named: wp_user_id

This attribute must contain the corresponding WordPress user ID.
